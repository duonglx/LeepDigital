API endpoint reference

POSTChallenge (login)
/api/v1/face-login/challenge
Headers: None (public endpoint, NO cookie)
Body: user_id, device_binding_id, action: "login"
Pre-flight: rate limit → combo key ACTIVE → not expired → fail_count < 5
Response: challenge_id, nonce, liveness_actions[]
Error codes: 429 (rate limited), 401 (not found / revoked / expired), 423 (locked)
POSTVerify (login)
/api/v1/face-login/verify
Headers: None (public endpoint)
Body: selfie (base64), challenge_id, nonce, user_id
Logic: validate challenge → Kalapa verify → score check → session create
On success: Set-Cookie: JSESSIONID, fail_count = 0, last_used_at = NOW()
On fail: fail_count++, if ≥ 5 → LOCK, return remaining_attempts
Response: 200 OK + user_info | 401 fail + remaining | 423 locked
