API endpoint reference

POSTChallenge
/api/v1/face-login/challenge
Headers: Cookie: JSESSIONID
Body: user_id, device_binding_id, action: "register"
Response: challenge_id, nonce, liveness_actions[]
TTL: 60 seconds, one-time nonce
POSTRegister
/api/v1/face-login/register
Headers: Cookie: JSESSIONID
Body: selfie (base64), challenge_id, nonce, liveness_data, device_binding_id
Logic: validate → Kalapa verify → REVOKE old → INSERT new
Response: 201 Created + user_display_info
POSTKalapa verify
/kalapa/v1/face/verify
Body: id_no (CCCD), selfie, liveness_data
Response: match_score, liveness_passed
Threshold: score ≥ 0.85 && liveness == true
Source: NFC chip CCCD indexed vectors
