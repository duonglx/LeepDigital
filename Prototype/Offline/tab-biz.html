Referral Business Flow

End-to-end journey from referral to commission payout

1. Referrer Journey (SOL App)

Open SOL App
Select 'My QR Biz Card'
System generate QR
Share via Social
Prospect Clicks Link / Scans QR

2. Prospect Journey (Landing Page)

Select Product
Enter Details
Anti-bot Check
Suspicious: Solve Captcha
Normal: Pass Verification
Success Screen
Landing Page API pushes Lead

3. Bank Operations (VCRM & Branches)

ZNS Confirmation
Telesale 'New Lead'
Call & Enrich
Route to Branch
RM 'Processing'
Closed Won
Batch / Daily Sync

4. Commission Payout (PBD System)

PBD extracts 'Closed Won'
Calculate Commission
Core Banking credits DDA
ZNS Reward Notice
